HW of lesson 2: Linear expressions

Task 1.
Given the width of the screen and the number of items to be displayed in a row. Determine the width of the items to be displayed in this row.
Task 2.
Enter the amount of salary and the subsistence minimum from the keyboard. Determine the amount of social assistance (assume that the salary is lower).
Task 3.
The age of the child is entered from the keyboard. To display in how many years she will visit the kindergarten, go to school, finish school, enter and finish university
Task 4.
Three valid variables are given a, b, and c. Create an algorithm that changes the values of these variables in places as follows: The variable a becomes the value of the variable b, the variable b becomes the value of the variable c, and the variable c becomes the previous value of the variable a.
Task 5.
Create an algorithm that calculates the whole and fractional part of the given number of x. For example: Number 23,56 => whole: 23, fractional: 0,56.
Task 6.
Develop a program (do not use if), which by year number Y defines the number of century C (consider that the beginning of XX century was 1901, not 1900).
Task 7.
The number of coins with a nominal value of 1 cent is entered from the keyboard. Determine how much $ and cents is in this amount . For example, given 245 cents, and output 2 $ 45 cents.
Task 8.
The current number of hours, minutes, seconds (as per clock) is given. Determine what time will be for the clock in N seconds
Task 9.
The current number of hours, minutes, seconds (as on hours). Determine how many seconds will be on the clock in N seconds. how many minutes will be on the clock in N seconds, X minutes.
Task 10.
Determine how many full weeks are in the specified number of days.
Task 11.
The number of the day of the week (1 to 7) is entered from the keypad. Determine which day of the week in N days.
Task 12.
The keypad is used to enter the number of months that have elapsed since a certain point in time. Bring the total number of years and months. For example, it is given 28 months, and let out 2 years and 4 months.
Task 13.
The month number is entered from the keypad. Determine which month will be after the specified number of N months.
Task 14.
Encrypt one entered English character in the upper case (one of the characters: A,B,…Z) by the shift method (Caesar's cipher).
