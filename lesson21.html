Lesson 21: DOM

Task 1.
Given 10 lines of the Hello! In individual div.When clicking on one of them, all subsequent should be painted red.
Task 2.
Given 5 input items.When entering the value in one of them, it is necessary to automatically fill in others (all previous in descending order each previous is 1 less than the next one), all subsequent on 1 more (each subsequent on 1 more than the previous one)
Task 3.
5 randomly generated numbered lists with random numbers were given (the number of elements in the list random from 1 to 10, the elements random - from 1 to 100).When you click on the button numbered lists with even number of elements, paint green, others in red.
Task 4.
3*3 with random numbers is given.If there is a click on any of the cells, a red frame is added to the appropriate table (try to add the number of clicks to the name of the table using the attribute added accordingly).
Task 5.
The form is introduced 7 numerical values (age, height, weight, wages, experience, department number, serial number).Add the CLICK event processing event to the form and if the click is on input, then automatically replace the value by 0.
