Lesson 20: DOM

Task 1.
Develop an item to enter the quantity of the selected product (minimum number 1, Maximum number 10, if exceeded block the button)
Task 2.
Dynamic search.There is a list of employees and a search box.When entering, all that contain the specified fragment are displayed
Task 3.
The list of cars (brand, year of release, price) is given.Form filtering items using a drop -down list (the content of these falling lists should be formed depending on the list transmitted).
Task 4.
Translator.The user is displayed mixed cards with words in English and Ukrainian.The user gradually clicks on cards (highlight the blue frame).If the correct pairs of cards corresponding to one word are found, we delete these cards.Otherwise - we select a red frame and in a second we pick up the frame.
Task 5.
One -line sapper.One -line table, which is added to the cells of the availability of a mine (use attributes).First, the cells are gray.When pressed on the cell, it analyzes whether there is a mine and then the color becomes red, if not green.Add the possibility of opening all the adjacent essential cells when opening an indispensable cell.
Task 6.
Develop a todo manager.The user enters the text of the problem and priority.The list of tasks added with the possibility of deleting problems and sorting by priority is displayed.
