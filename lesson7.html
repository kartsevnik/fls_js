Lesson 7: Function

Task 0.
Create a function that returns the time of year to which this month belongs by the month number.
Task 1.
Create a function that returns the name of the month by the month number.
Task 2.
Create a function that allows you to find out whether this day is a working day by the day number.
Task 3.
Create separate functions that find for 4 numbers:

1)the sum;

2) product;

3) arithmetic mean;

4) the minimum value.

Task 4.
Create a function that for 3 given numbers finds several results at the same time: the number of even numbers, the number of positive numbers, the number greater than 100.
Task 5.
Create separate functions that translate:

Centimeters to inches;

Kilograms to pounds;

Kilometers to miles.

Task 6.
Create a function that creates a table with a specified number of rows and columns (the table is filled with a given fixed message).
Task 7.
Create a function that randomly displays one of 4 images (paths to the images are passed to the function)
Task 8.
Create a function that displays a banner (the following are passed to the function: image, title and link to go to when you click on the image (the img tag should be in the middle of the a tag:

<a> <img src="path"> </a>).

Task 9.
Temperature readings are given (an arbitrary number). Develop a function that will allow you to count the number of negative temperature readings.
Task 10.
Temperature readings are given (an arbitrary number). Develop a function that will find the average value for positive temperature readings.
