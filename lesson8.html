Lesson 8: Array

Task 1.
An array containing grades in K subjects is given. Find the average grade and find out what category it belongs to (excellent student, "D" student (has at least one "D"), "A" student (grades "good" and "excellent"), "C" student (has at least one "C")).
Task 2.
Given an array that stores the number of visitors to the store during the week.
Task 3.
An array of names of students is given. Find out how many times the name "Ivan" appears.
Task 4.
The sequence of car license plates is given. Count the number of license plates that :

- begin with the letter "A";

- the first and the last letters are the same;

- consist of more than 5 characters;

Task 5.
A sequence of student grades is given. Calculate the number of:

1. deuces

2. the number of good grades (good, excellent);

3. the number of grades that are below average.

Task 6.
Given a sequence of product prices and names (in separate arrays). Display those that the user can afford (the amount of money is specified).
Task 7.
The sequence of payments during the year is given. Find the total amount of money for different periods
Task 8.
Given a one-dimensional array that stores a certain winning amount (the elements are randomly filled with values from -500 to 500). Allow the user to choose the numbers of elements (until he refuses). Find the total winning amount.
Task 9.
Naval battle. The user enters the number of cells of a one-dimensional array and the number of single ships. The computer randomly places these single ships in the array, one per cell (if there is 0 in a cell, the cell is empty, if there is 1, it means that there is a ship there). The user enters the number of the cell where he/she is shooting. The game continues until all ships are sunk.
