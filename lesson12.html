Lesson 12: Array and sort

Task 0.
Arrange so that first there are wins/losses (i.e., modulo) less than 10, then less than 100, then less than 200, then less than 300, then less than 400, then less than or equal to 500
Task 1-3.
An array of 30 random integers is given. Count the number of exchanges during the bubble sorting, the shuffle sort and the inclusion sort.
Task 4.
For the considered sorting methods, try to output the sorting stages by outputting the corresponding tables according to the sample.
Task 5.
An array of names is given. Using the appropriate sorting and binary search, determine whether the name "Olga" is in the array and at what index.
Task 6.
An array of names is given. Using the appropriate sorting and binary search, determine if there is a name in the array with a length of 5 characters and at what index.
