Lesson 13: Recursion and Looping

Task 1.
Let x0 = 1, x i = xi-1 + 2i, where i = 1,2,.... Determine x10. Solve using recursive and nonrecursive algorithms
Task 2.
Let x0 = x1 = 1, xi = xi-1 + 2xi-2, where i = 2,3,.... Determine xn. Solve using recursive and nonrecursive algorithms
Task 3.
Using closures, develop an iterator, that is, a function that will gradually produce one value from the specified minimum to the specified maximum in separate calls. If the value reaches the maximum, the next value will be the minimum. Using this function, implement an iteration of the month numbers.
Task 4.
Using closures, develop an iterator, i.e. a function that will gradually produce one letter of a predetermined word on separate calls. Use it to develop the game "Translator". We display a description of the word for the user and the user enters the translation of the word one letter at a time. Count the number of guessed letters.
Task 5.
Addition simulator. Every 10 seconds, the user is given a random example of adding two digits and tested.
