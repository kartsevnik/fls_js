Recursion and Looping: Task 3

Task 3. З використанням замикань розробити ітератор, тобто функцію, що буде поступово за окремими викликами видавати по одне значення від заданого мінімального до заданого максимального. Якщо значення досягне максимального, то наступним буде мінімальне значення. З використанням цієї функції реалізувати перебір номерів місяців.

Task 3. Using closures, develop an iterator, that is, a function that will gradually produce one value from the specified minimum to the specified maximum in separate calls. If the value reaches the maximum, the next value will be the minimum. Using this function, implement an iteration of the month numbers.
