Recursion and Looping: Task 4

Task 4. З використанням замикань розробити ітератор, тобто функцію, що буде поступово за окремими викликами видавати по одну букву наперед заданого слова. З використанням нього розробити гру «Прекладач». Виводимо для користувача опис цього слова і по одну букву користувач вводить переклад цього слова. Підрахувати кількість вгаданих букв.

Task 4. Using closures, develop an iterator, i.e. a function that will gradually produce one letter of a predetermined word on separate calls. Use it to develop the game "Translator". We display a description of the word for the user and the user enters the translation of the word one letter at a time. Count the number of guessed letters.
