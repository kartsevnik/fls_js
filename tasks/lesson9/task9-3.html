Array: Task 3.

Користувач вводить кількість елементів. Створити масив, у якому перші 5 елементів заповнені 1-цями, а інші до кінця масиву заповнені 7-ками.

The user enters the number of elements. Create an array in which the first 5 elements are filled with 1s, and the rest are filled with 7s until the end of the array.
