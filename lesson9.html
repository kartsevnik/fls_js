Lesson 9: Array and Function

Task 1.
The user enters the number of elements. Create an array consisting of the specified number of elements filled with zeros.
Task 2.
The user enters the number of elements. Create an array in which the first half is filled with 1s and the second half is filled with 7s.
Task 3.
The user enters the number of elements. Create an array in which the first 5 elements are filled with 1s, and the rest are filled with 7s until the end of the array.
Task 4.
An array of elements is given. Print the elements greater than 100
Task 5.
An array of elements is given. Find the product of positive elements.
Task 6.
An array of elements is given. Multiply all elements that are larger than the first element by 2
Task 7.
An array of prices is given. Change this array so that the prices of goods that are more than 1000 UAH. give a 30% discount.
Task 8.
An array of car license plates is given. Create a new array quietly, starting with "A"
Task 9.
An array of names is given. Create an array of the first letters of these names.
Task 10.
An array of prices in UAH is given. The tax is 20%. Create an array containing the amount of tax paid in UAH.
